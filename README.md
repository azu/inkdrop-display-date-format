## Installation

    ipm install inkdrop-display-date-format

## Setting

You can change date format on Setting Panel.

- Default: `YYYY-MM-DD`
- Available format: <https://day.js.org/docs/en/parse/string-format#list-of-all-available-parsing-tokens>
